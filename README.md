# EventHub: Centralized Event Management System

## Introduction
EventHub is a comprehensive solution designed to address the inefficiencies in managing academic and professional events[^1^][1]. By integrating cutting-edge technologies, EventHub offers a seamless, interactive, and efficient experience for both organizers and attendees[^2^][2].

## Problem Statement
The traditional methods of managing event information are often fragmented and outdated. EventHub aims to centralize the process, providing real-time updates and consistent data across various platforms.

## Objectives
- Develop a centralized platform for event management[^3^][3].
- Ensure secure user authentication and login[^4^][4][^5^][5].
- Simplify administrative tasks for operational efficiency[^6^][6].
- Maintain data integrity with validation for data entry[^7^][7].
- Create an intuitive form for entering event details[^8^][8].

## System Design
EventHub utilizes the MERN stack, combining MongoDB, Express.js, React, and Node.js to create a robust and responsive web application[^9^][9].

## Modules
- **Registration Module**: Manages user accounts and secure authentication[^10^][10].
- **Conference Booking Module**: Enables faculty to book conferences seamlessly[^11^][11].
- **Event Display Module**: Presents information about upcoming conferences[^12^][12].
- **Calendar Module**: Provides a visual representation of scheduled events[^13^][13].

## Testing
The system has been rigorously tested using Selenium IDE to ensure functionality and user experience.

## Conclusion & Future Scope[^14^][14]
EventHub revolutionizes event management with its digital platform. Future enhancements may include AR/VR integration, advanced analytics, and a dedicated mobile application.

## Installation
To set up EventHub on your local machine, follow these steps:
1. Clone the repository.
2. Install the required dependencies.
3. Set up the local server environment using WampServer.
4. Run the application and navigate to the localhost in your browser.
